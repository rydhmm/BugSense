# 🐛 BugSense — C Language Error Detector

A simplified compiler front-end that detects and explains common C programming errors.

## Features
- 🔵 Lexical Analysis — detects invalid characters, unterminated strings
- 🔴 Syntax Analysis — detects missing semicolons, unmatched brackets
- 🟣 Semantic Analysis — detects undeclared variables, type mismatches

## Tech Stack
- **Backend:** Java, Spring Boot
- **Frontend:** HTML, CSS, JavaScript
- **Architecture:** REST API (Client-Server)

## How to Run
1. Clone the repo
2. Run backend:
   cd backend
   mvn spring-boot:run
3. Open frontend/index.html in browser
